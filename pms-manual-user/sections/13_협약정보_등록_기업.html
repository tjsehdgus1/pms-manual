협약정보 등록 사용자

메뉴 경로: 마이페이지 > 과제수행 > 전자협약

1. 전자협약 과제 목록 진입

마이페이지 「과제수행」 → 「전자협약」 클릭 → 해당 과제 행 클릭

2. 협약 목록 확인

메뉴 경로: 전자협약 > 협약목록

과제의 협약 목록을 확인하고 해당 협약 행을 클릭합니다.

3. 요청내용 확인

메뉴 경로: 협약상세 > 요청내용

관리자가 작성한 협약정보 등록 요청 내용을 확인합니다.

4. 과제정보 입력

메뉴 경로: 협약상세 > 과제정보

과제명·연구기간·주관기관 등 과제 기본정보를 입력하고 「저장」 클릭

5. 컨소시엄 정보 입력 (해당 시)

메뉴 경로: 협약상세 > 컨소시엄

컨소시엄 참여기관 정보를 확인하고 필요 시 입력합니다.

6. 사업비 입력

메뉴 경로: 협약상세 > 사업비

정부지원금·자부담 등 사업비 항목을 입력하고 「저장」 클릭

7. 참여연구원 등록

메뉴 경로: 협약상세 > 참여연구원

참여연구원 목록을 확인하고 「참여인력 추가」 버튼을 클릭하여 인원을 추가합니다.

참여인력 추가

팝업에서 성명·소속·역할 등을 확인하고 추가할 인원의 행에 대해 「선택」 클릭

8. 첨부자료 등록

메뉴 경로: 협약상세 > 첨부자료

협약에 필요한 서류를 첨부하고 「저장」 클릭 → 전체 입력 완료 후 「최종제출」 버튼 클릭

💡 안내
협약정보 등록 요청 완료 후 관리자가 내용을 검토하고 협약서를 등록합니다.
