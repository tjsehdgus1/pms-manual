# 전남테크노파크 PMS 사용자 매뉴얼

# 사업관리

**메뉴 경로:** 관리자 > 사업등록관리 > 사업관리

---

**1. 사업 목록 진입**
상단 메뉴 「사업등록관리」 → 왼쪽 메뉴 「사업관리」 클릭

[화면 캡처 : 사업관리 목록 화면]

**2. 「등록」 버튼 클릭**
목록 오른쪽 하단 「등록」 클릭

> ⚠️ 기존 사업 행 클릭 시 수정 화면으로 이동합니다.

**3. 사업 정보 입력**

| 영역 | 주요 항목 |
|------|----------|
| ① 사업정보 | 사업명, 부처, 전담기관, 연구과제번호, 국가과학기술분류, 연구과제명, 사업비 |
| ② 주관기관 정보 | 책임자·실무자 정보, 기관명, 사업자·법인등록번호, 주소 |
| ③ 공동연구개발기관 | 참여기관 해당 여부, 기관 추가, 첨부파일 (최대 5개) |

[화면 캡처 : 사업 등록 입력 화면]

> ⚠️ ★ 표시 항목은 필수입니다.

**4. 「등록」으로 저장**
오른쪽 하단 「등록」 클릭 → 저장 완료 후 목록으로 자동 이동

> ⚠️ 이후 **프로젝트 관리**에서 프로젝트를 등록해야 공고 등록이 가능합니다.

---

# 프로젝트 관리

**메뉴 경로:** 관리자 > 사업등록관리 > 프로젝트 관리

---

**1. 프로젝트 목록 진입**
왼쪽 메뉴 「프로젝트 관리」 클릭

[화면 캡처 : 프로젝트 관리 목록 화면]

**2. 「프로젝트 등록」 버튼 클릭**
목록 오른쪽 하단 「프로젝트 등록」 클릭

**3. 사업 선택**
상단 「사업 조회」 버튼 클릭 → 등록된 사업 선택

> ⚠️ 사업이 먼저 등록되어 있어야 합니다.

**4. 프로젝트 정보 입력**

| 항목 | 설명 |
|------|------|
| 프로젝트번호 | 자동 생성 |
| 프로젝트명 | 직접 입력 |
| 프로젝트기간 | 시작일 ~ 종료일 선택 |
| 목표건수 | 직접 입력 |
| 예산 | 직접 입력 |

[화면 캡처 : 프로젝트 등록 입력 화면]

> ⚠️ ★ 표시 항목은 필수입니다.

**5. 「등록」으로 저장**
오른쪽 하단 「등록」 클릭 → 저장 완료 후 목록으로 자동 이동

> ⚠️ 이후 **지원사업공고**에서 공고를 등록해야 기업이 신청할 수 있습니다.

# 지원사업공고

**메뉴 경로:** 관리자 > 사업등록관리 > 지원사업공고

---

**1. 지원사업공고 목록 진입**
왼쪽 메뉴 「지원사업공고」 클릭

![지원사업공고 목록 화면](../images/02_지원사업공고/02_지원사업공고_목록_화면.png)

> ⚠️ 목록의 공고 행 클릭 시 수정 화면으로 이동합니다. 신규 등록 시 오른쪽 하단 「등록」 버튼을 클릭하세요.

**2. 「등록」 버튼 클릭**
목록 오른쪽 하단 「등록」 클릭 → 공고 등록 화면으로 이동

**3. 프로젝트 선택**
상단 「프로젝트 조회」 버튼 클릭 → 팝업에서 해당 프로젝트 행의 「선택」 클릭

![프로젝트 조회 팝업](../images/02_지원사업공고/02_프로젝트_조회_팝업.png)

> ⚠️ 프로젝트가 먼저 등록되어 있어야 목록에 표시됩니다.

**4. 공고기본 정보 입력**

| 항목 | 필수 | 설명 |
|------|------|------|
| 공고번호 | | 저장 시 자동 생성 |
| 공고관리 주체 구분 | ★ | 테크노파크 PMS / 타사(IRIS, SMTECH 등) 선택 |
| 공고명 | ★ | 직접 입력 |
| 지원유형 | ★ | R&D / 비R&D 선택 |
| 과제분류 | ★ | 단독과제 / 전소시험과제 선택 |
| 표출여부 | ★ | 표출 / 숨김 선택 |
| 과제적격성진단표 | ★ | 「조회」 버튼으로 선택 |
| 공고기간 | ★ | 시작일 ~ 종료일 선택 |
| 접수기간 | ★ | 시작일 ~ 종료일 선택 |
| 담당자 | ★ | 「담당자 조회」로 검색 후 선택 |
| 공고내용 | | 에디터로 자유 입력 |
| 공고문 첨부 | | 파일 1개 첨부 |
| 첨부파일 | | 파일 최대 5개 첨부 |
| 요청 서류 관리 | | 「항목 추가」로 제출 서류 목록 구성 |

![지원사업공고 등록 입력 화면](../images/02_지원사업공고/02_지원사업공고_등록_입력_화면.png)

> ⚠️ 공고관리 주체를 **타사**로 선택하면 타사 접수 링크 URL 입력란이 활성화됩니다.

**5. 「등록」으로 저장**
오른쪽 하단 「등록」 클릭 → 저장 완료 후 목록으로 자동 이동

**6. 프로그램 목록 확인**
저장 후 공고 상세 화면 상단의 「프로그램 정보」 탭 클릭

![프로그램 목록 화면](../images/02_지원사업공고/02_프로그램_목록_화면.png)

**7. 프로그램 등록**
「프로그램 추가」 클릭 → 프로그램 정보 입력 후 「등록」

![프로그램 등록 입력 화면](../images/02_지원사업공고/02_프로그램_등록_입력_화면.png)

> ⚠️ 프로그램이 등록되어야 기업이 해당 공고로 과제를 신청할 수 있습니다.

# 공고 신청 (기업)

**메뉴 경로:** 홈 > 사업공고 > 지원사업 공고

---

**1. 지원사업 공고 목록 진입**
상단 메뉴 「사업공고」 → 「지원사업 공고」 클릭

![지원사업 공고 목록 화면](../images/03_공고신청_기업/03_지원사업_공고_목록_화면.png)

**2. 공고 상세 확인**
신청할 공고 행 클릭 → 공고 상세 화면으로 이동

![지원사업 공고 상세 화면](../images/03_공고신청_기업/03_지원사업_공고_상세_화면.png)

**3. 프로그램 신청**
공고 상세 화면에서 신청할 프로그램의 「신청」 버튼 클릭

![프로그램 신청 버튼](../images/03_공고신청_기업/03_프로그램_신청_버튼.png)

> ⚠️ 접수기간 내에만 신청 버튼이 활성화됩니다. 로그인 후 마이페이지 사업공고접수 메뉴로 이동합니다.

**4. Step 1 – 프로그램정보 확인**
사업공고명·프로그램명·지원분야·총사업비 등 신청 정보 확인 (읽기 전용)

![프로그램정보 화면](../images/03_공고신청_기업/03_프로그램정보_화면.png)

**5. Step 2 – 기업정보 입력**
임직원 현황·재무정보(3개년)·사업자등록증 등 입력 후 「저장」

![기업정보 입력 화면](../images/03_공고신청_기업/03_기업정보_입력_화면.png)

> ⚠️ ★ 표시 항목은 필수입니다.

**6. Step 3 – 적격성진단표 작성**
항목별 Y / N 선택 후 「저장」

![적격성진단표 화면](../images/03_공고신청_기업/03_적격성진단표_화면.png)

> ⚠️ 모든 항목에 응답해야 저장됩니다.

**7. Step 4 – 컨소시엄 (해당 시)**
단독 신청인 경우 건너뜁니다. 컨소시엄 신청 시 「기업(기관) 추가」 클릭 → 팝업에서 검색 후 선택

![컨소시엄 화면](../images/03_공고신청_기업/03_컨소시엄_화면.png)

![기업(기관) 추가 팝업](../images/03_공고신청_기업/03_기업_기관_추가_팝업.png)

> ⚠️ 참여기관 담당자가 직접 로그인하여 기업정보 입력 후 참여완료 처리해야 합니다.

**8. Step 5 – 사업정보 입력**
과제명·연구기간·예산·사업계획서 파일 입력 후 「저장」

![사업정보 입력 화면](../images/03_공고신청_기업/03_사업정보_입력_화면.png)

> ⚠️ 사업계획서 파일은 최대 100MB.

**9. 접수완료**
모든 단계 작성 완료 후 「프로그램정보」 탭으로 이동 → 「접수완료」 버튼 클릭

![프로그램정보 – 접수완료 화면](../images/03_공고신청_기업/03_프로그램정보_접수완료_화면.png)

> ⚠️ **접수완료** 버튼을 클릭해야 최종 접수됩니다. 접수 후에는 내용 수정이 불가합니다.

# 요건검토 / 보완요청

**메뉴 경로:** 관리자 > 과제관리 > 공고접수내역

---

**1. 공고접수내역 목록 진입**
왼쪽 메뉴 「과제관리」 → 「공고접수내역」 클릭

![공고목록 화면](../images/06_보완요청_답변검토_관리자/06_공고목록_화면.png)

공고 행 클릭 → 접수목록에서 검토할 신청서 행 클릭

![접수목록 화면](../images/06_보완요청_답변검토_관리자/06_접수목록_화면.png)

**2. 프로그램정보 확인**
사업공고명·프로그램명·지원분야·총사업비 등 신청 정보를 확인합니다.

![프로그램정보 확인 화면](../images/04_요건검토_보완요청/04_프로그램정보_확인_화면.png)

**3. 기업정보 확인**
기업이 제출한 임직원 현황·재무정보·사업자등록증 등을 검토합니다.

![기업정보 확인 화면](../images/04_요건검토_보완요청/04_기업정보_확인_화면.png)

**4. 적격성진단표 확인**
기업이 제출한 항목별 Y / N 응답 내용을 검토합니다.

![적격성진단표 확인 화면](../images/04_요건검토_보완요청/04_적격성진단표_확인_화면.png)

**5. 컨소시엄 확인 (해당 시)**
참여기관 정보 및 참여완료 여부를 확인합니다.

![컨소시엄 확인 화면](../images/04_요건검토_보완요청/04_컨소시엄_확인_화면.png)

**6. 사업정보 확인**
과제명·연구기간·예산·사업계획서 파일 등을 검토합니다.

![사업정보 확인 화면](../images/04_요건검토_보완요청/04_사업정보_확인_화면.png)

**7. 요건검토 체크리스트 작성**
항목별 적합 / 부적합 선택 후 「저장」 클릭

![요건검토 체크리스트 화면](../images/04_요건검토_보완요청/04_요건검토_체크리스트_화면.png)

> ⚠️ 모든 항목에 응답해야 저장됩니다.

**8. 보완요청 (필요 시)**
보완이 필요한 내용을 입력하고 「보완요청」 버튼 클릭

![요건검토 보완요청 화면](../images/04_요건검토_보완요청/04_요건검토_보완요청_화면.png)

> ⚠️ 보완요청 후 기업 담당자가 보완 내용을 확인하고 답변을 제출해야 합니다.

# 보완요청 답변

**메뉴 경로:** 마이페이지 > 접수관리 > 사업공고접수

---

**1. 사업공고접수 목록 진입**
상단 메뉴 「마이페이지」 → 「접수관리」 → 「사업공고접수」 클릭 → 해당 접수 건 행 클릭

![사업공고접수 목록 화면](../images/05_보완요청_답변_기업/05_사업공고접수_목록_화면.png)

**2. 요건검토 탭 확인**
접수 상세 화면에서 「요건검토」 탭 클릭 → 관리자 검토 결과 및 보완요청 여부 확인

![요건검토 탭 화면](../images/06_보완요청_답변검토_관리자/06_요건검토_탭_화면.png)

**3. 보완요청 확인 및 답변 작성**
보완요청 내용 확인 후 답변 내용 입력 → 「답변 제출」 버튼 클릭

![보완요청 확인 및 답변 화면](../images/05_보완요청_답변_기업/05_보완요청_확인_및_답변_화면.png)

> ⚠️ 답변 제출 후에는 내용 수정이 불가합니다. 관리자가 답변을 검토 후 처리합니다.

# 보완요청 답변 검토

---

**1. 공고접수내역 목록 진입**
왼쪽 메뉴 「과제관리」 → 「공고접수내역」 클릭

![공고목록 화면](../images/06_보완요청_답변검토_관리자/06_공고목록_화면.png)

공고 행 클릭 → 접수목록에서 해당 신청서 행 클릭

![접수목록 화면](../images/06_보완요청_답변검토_관리자/06_접수목록_화면.png)

**2. 요건검토 탭 확인**
접수 상세 화면에서 「요건검토」 탭 클릭 → 기업 보완 답변 제출 여부 확인

![요건검토 탭 화면](../images/06_보완요청_답변검토_관리자/06_요건검토_탭_화면.png)

![제출 내용보기 팝업 화면](../images/23_보고서_제출확인_관리자/23_제출_내용보기_팝업_화면.png)

> 💡 보고서 제출 내용 확인 후 다음 단계인 중간/결과 평가 등록으로 진행합니다.
