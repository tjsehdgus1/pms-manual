# 전남테크노파크 PMS 사용자 매뉴얼

# 사업관리

**메뉴 경로:** 관리자 > 사업등록관리 > 사업관리

---

**1. 사업 목록 진입**
상단 메뉴 「사업등록관리」 → 왼쪽 메뉴 「사업관리」 클릭

[화면 캡처 : 사업관리 목록 화면]

**2. 「등록」 버튼 클릭**
목록 오른쪽 하단 「등록」 클릭

> ⚠️ 기존 사업 행 클릭 시 수정 화면으로 이동합니다.

**3. 사업 정보 입력**

| 영역 | 주요 항목 |
|------|----------|
| ① 사업정보 | 사업명, 부처, 전담기관, 연구과제번호, 국가과학기술분류, 연구과제명, 사업비 |
| ② 주관기관 정보 | 책임자·실무자 정보, 기관명, 사업자·법인등록번호, 주소 |
| ③ 공동연구개발기관 | 참여기관 해당 여부, 기관 추가, 첨부파일 (최대 5개) |

[화면 캡처 : 사업 등록 입력 화면]

> ⚠️ ★ 표시 항목은 필수입니다.

**4. 「등록」으로 저장**
오른쪽 하단 「등록」 클릭 → 저장 완료 후 목록으로 자동 이동

> ⚠️ 이후 **프로젝트 관리**에서 프로젝트를 등록해야 공고 등록이 가능합니다.

---

# 프로젝트 관리

**메뉴 경로:** 관리자 > 사업등록관리 > 프로젝트 관리

---

**1. 프로젝트 목록 진입**
왼쪽 메뉴 「프로젝트 관리」 클릭

[화면 캡처 : 프로젝트 관리 목록 화면]

**2. 「프로젝트 등록」 버튼 클릭**
목록 오른쪽 하단 「프로젝트 등록」 클릭

**3. 사업 선택**
상단 「사업 조회」 버튼 클릭 → 등록된 사업 선택

> ⚠️ 사업이 먼저 등록되어 있어야 합니다.

**4. 프로젝트 정보 입력**

| 항목 | 설명 |
|------|------|
| 프로젝트번호 | 자동 생성 |
| 프로젝트명 | 직접 입력 |
| 프로젝트기간 | 시작일 ~ 종료일 선택 |
| 목표건수 | 직접 입력 |
| 예산 | 직접 입력 |

[화면 캡처 : 프로젝트 등록 입력 화면]

> ⚠️ ★ 표시 항목은 필수입니다.

**5. 「등록」으로 저장**
오른쪽 하단 「등록」 클릭 → 저장 완료 후 목록으로 자동 이동

> ⚠️ 이후 **지원사업공고**에서 공고를 등록해야 기업이 신청할 수 있습니다.

# 지원사업공고

**메뉴 경로:** 관리자 > 사업등록관리 > 지원사업공고

---

**1. 지원사업공고 목록 진입**
왼쪽 메뉴 「지원사업공고」 클릭

[화면 캡처 : 지원사업공고 목록 화면]

> ⚠️ 목록의 공고 행 클릭 시 수정 화면으로 이동합니다. 신규 등록 시 오른쪽 하단 「등록」 버튼을 클릭하세요.

**2. 「등록」 버튼 클릭**
목록 오른쪽 하단 「등록」 클릭 → 공고 등록 화면으로 이동

**3. 프로젝트 선택**
상단 「프로젝트 조회」 버튼 클릭 → 팝업에서 해당 프로젝트 행의 「선택」 클릭

[화면 캡처 : 프로젝트 조회 팝업]

> ⚠️ 프로젝트가 먼저 등록되어 있어야 목록에 표시됩니다.

**4. 공고기본 정보 입력**

| 항목 | 필수 | 설명 |
|------|------|------|
| 공고번호 | | 저장 시 자동 생성 |
| 공고관리 주체 구분 | ★ | 테크노파크 PMS / 타사(IRIS, SMTECH 등) 선택 |
| 공고명 | ★ | 직접 입력 |
| 지원유형 | ★ | R&D / 비R&D 선택 |
| 과제분류 | ★ | 단독과제 / 전소시험과제 선택 |
| 표출여부 | ★ | 표출 / 숨김 선택 |
| 과제적격성진단표 | ★ | 「조회」 버튼으로 선택 |
| 공고기간 | ★ | 시작일 ~ 종료일 선택 |
| 접수기간 | ★ | 시작일 ~ 종료일 선택 |
| 담당자 | ★ | 「담당자 조회」로 검색 후 선택 |
| 공고내용 | | 에디터로 자유 입력 |
| 공고문 첨부 | | 파일 1개 첨부 |
| 첨부파일 | | 파일 최대 5개 첨부 |
| 요청 서류 관리 | | 「항목 추가」로 제출 서류 목록 구성 |

[화면 캡처 : 지원사업공고 등록 입력 화면]

> ⚠️ 공고관리 주체를 **타사**로 선택하면 타사 접수 링크 URL 입력란이 활성화됩니다.

**5. 「등록」으로 저장**
오른쪽 하단 「등록」 클릭 → 저장 완료 후 목록으로 자동 이동

> ⚠️ 이후 **공고 신청(기업)** 단계에서 기업이 해당 공고를 확인하고 신청할 수 있습니다.
