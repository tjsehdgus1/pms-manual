사업비 지급 처리 관리자

메뉴 경로: 관리자 > 과제관리 > 사업비지급신청

1. 사업비지급신청 공고 목록 진입

왼쪽 메뉴 「과제관리」 → 「사업비지급신청」 클릭 → 해당 공고 행 클릭

2. 프로그램 / 과제 목록 확인

메뉴 경로: 사업비지급신청 > 프로그램/과제 목록

지급 신청이 접수된 과제 목록을 확인하고 해당 과제 행을 클릭합니다.

3. 사업비 지급신청 정보 확인

메뉴 경로: 사업비지급신청 > 신청정보

기업이 신청한 지급 내역·계좌정보 등을 검토하고 상세 내용을 확인합니다.

4. 사업비 지급신청 상세 확인 및 처리

메뉴 경로: 사업비지급신청 정보 > 신청 상세 (팝업)

팝업에서 신청 항목별 상세 내용을 확인하고 「지급결의」 클릭 → 실제 사업비 지급 후 「지급완료」 버튼 클릭

💡 안내
지급처리 완료 후 기업은 사업비 집행관리 메뉴에서 집행 내역을 등록할 수 있습니다.
