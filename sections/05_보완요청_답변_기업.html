보완요청 답변 기업

메뉴 경로: 마이페이지 > 접수관리 > 사업공고접수

1. 사업공고접수 목록 진입

상단 메뉴 「마이페이지」 → 「접수관리」 → 「사업공고접수」 클릭 → 해당 접수 건 행 클릭

2. 요건검토 탭 확인

메뉴 경로: 사업공고접수 > 요건검토

접수 상세 화면에서 「요건검토」 탭 클릭 → 관리자 검토 결과 및 보완요청 여부 확인

3. 보완요청 확인 및 답변 작성

메뉴 경로: 사업공고접수 > 요건검토 > 보완요청 확인 및 답변

보완요청 내용 확인 후 답변 내용 입력 → 「답변 제출」 버튼 클릭

⚠️ 주의사항
답변 제출 후에는 내용 수정이 불가합니다. 관리자가 답변을 검토 후 처리합니다.
