회원가입 사용자

메뉴 경로: 홈 > 회원가입

1. 회원 유형 선택

홈 화면에서 「회원가입」 클릭 → 가입 유형 선택

💡 가입 유형 안내
유형 대상
일반회원 개인 자격으로 가입하는 경우
기업회원 기업·기관 소속으로 사업에 신청하는 경우 (권장)

A. 일반회원 가입

A-1. 약관 동의

이용약관·개인정보 처리방침을 확인하고 전체 동의 후 「다음」 클릭

A-2. 본인인증

휴대폰 본인인증 진행 → 인증번호 입력 후 확인

A-3. 회원정보 작성

아이디·비밀번호·이메일 등 기본 정보를 입력하고 「가입완료」 클릭

B. 기업회원 가입

💡 안내
사업 신청을 위해서는 기업회원으로 가입해야 합니다. 기업 정보를 먼저 등록하거나 조회한 후 회원 가입이 진행됩니다.

B-1. 약관 동의

이용약관·개인정보 처리방침을 확인하고 전체 동의 후 「다음」 클릭

B-2. 기업(기관) 조회 / 등록 선택

소속 기업이 이미 PMS에 등록되어 있으면 「기업조회」, 처음 등록하는 경우 「기업등록」 선택

B-3. 기업 조회

사업자등록번호 또는 기업명으로 검색 → 해당 기업 선택

💡 안내
조회 결과에 기업이 없으면 「기업등록」으로 진행하세요. (아래 B-4 ~ B-5 단계)

B-4. 공동인증서 인증 (기업 신규 등록 시)

기업을 처음 등록하는 경우 공동인증서로 인증합니다. 「인증서 선택」 후 비밀번호 입력

B-5. 기업(기관) 정보 작성 (기업 신규 등록 시)

기업명·사업자등록번호·주소·대표자 등 기업 정보를 입력하고 「등록」 클릭

⚠️ 주의
사업자등록증과 동일한 정보로 입력해야 합니다. 잘못 입력하면 이후 협약·정산 단계에서 오류가 발생할 수 있습니다.

B-6. 본인인증

휴대폰 본인인증 진행 → 인증번호 입력 후 확인

B-7. 회원정보 작성

아이디·비밀번호·이메일·소속 부서 등 정보를 입력하고 「가입완료」 클릭

💡 안내
가입 완료 후 로그인하면 PMS 메인 화면으로 이동합니다. 이후 공고 신청으로 진행하세요.
