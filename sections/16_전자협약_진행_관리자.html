전자협약 진행 관리자

메뉴 경로: 관리자 > 과제관리 > 전자협약

1. 전자협약 공고 목록 진입

왼쪽 메뉴 「과제관리」 → 「전자협약」 클릭 → 해당 공고 행 클릭

2. 프로그램 / 과제 목록 확인

메뉴 경로: 전자협약 > 프로그램/과제 목록

기업이 전자협약 요청한 과제 목록을 확인하고 해당 과제 행을 클릭합니다.

3. 협약 목록 확인

메뉴 경로: 전자협약 > 협약목록

협약 진행 상태를 확인하고 해당 협약 행을 클릭합니다.

4. 전자협약 내용 확인

메뉴 경로: 전자협약 > 전자협약진행

기업의 전자서명 완료 현황을 확인하고 관리자 서명을 진행합니다. 「공동인증서 인증」 버튼 클릭

5. 공동인증서 인증 (담당기관 전자서명)

메뉴 경로: 전자협약진행 > 공동인증서 인증

담당기관(전남테크노파크) 명의의 공동인증서를 선택하고 비밀번호를 입력하여 전자서명을 완료합니다.

6. TSA 인증마크 확인

메뉴 경로: 전자협약진행 > TSA 인증마크 확인

전자서명 완료 후 TSA(타임스탬프) 인증마크를 확인하여 협약 체결을 최종 검증합니다.

💡 안내
TSA 인증마크 확인 완료 시 전자협약이 최종 체결됩니다. 이후 사업비 지급 신청 단계로 진행합니다.
