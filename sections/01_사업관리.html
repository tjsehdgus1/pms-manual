01. 사업관리

1-1. 사업 등록

신규 사업을 시스템에 등록하는 절차입니다. 메인 메뉴에서 「사업관리」 → 「사업 등록」 을 클릭합니다.

🖼
[화면 캡처 : 사업 등록 화면]
사업 등록 폼 전체 화면

입력 항목

항목 필수 설명
사업명●사업의 공식 명칭을 입력합니다.
사업 기간●시작일 ~ 종료일을 선택합니다.
총 사업비●정부지원금 + 민간부담금 합계를 입력합니다.
사업 유형●일반 / R&D / 인프라 중 선택합니다.
담당자○사업 담당 직원을 검색하여 지정합니다.
비고○특이사항을 자유롭게 입력합니다.

등록 절차

  1. 상단 메뉴 「사업관리」 클릭
  2. 좌측 메뉴 「사업 등록」 클릭
  3. 각 항목 입력 후 「저장」 클릭
  4. 저장 완료 알림 확인 → 사업 목록으로 자동 이동
⚠️ 주의사항
총 사업비는 실제 협약금액과 일치해야 합니다. 잘못 입력한 경우 「사업 수정」 메뉴에서 담당자 승인 후 변경할 수 있습니다.
🖼
[화면 캡처 : 사업 등록 완료 화면]
저장 성공 메시지 및 사업 목록 리다이렉트

1-2. 사업 조회 / 수정

등록된 사업을 조회하고 내용을 수정하는 방법입니다. 「사업관리」 → 「사업 목록」 에서 해당 사업을 클릭하면 상세 화면으로 이동합니다.

🖼
[화면 캡처 : 사업 목록 화면]
검색 필터 및 목록 테이블

검색 필터 사용법

  1. 사업명, 기간, 상태 등 원하는 조건 입력
  2. 「검색」 버튼 클릭
  3. 결과 목록에서 사업명 클릭 → 상세 화면 이동
  4. 수정이 필요하면 「수정」 버튼 클릭
  5. 내용 변경 후 「저장」 클릭
⚠️ 주의사항
승인 완료된 사업은 일부 항목(총 사업비, 사업 기간 등)이 잠금 처리됩니다. 변경이 필요한 경우 관리자에게 잠금 해제를 요청하세요.
💡 참고
목록 우측 상단의 Excel 다운로드 기능으로 검색된 사업 목록 전체를 엑셀 파일로 내보낼 수 있습니다.
🖼
[화면 캡처 : 사업 상세 수정 화면]
수정 모드 활성화 상태

1-3. 사업 승인

담당자가 등록한 사업을 관리자가 최종 승인하는 절차입니다. 승인 권한이 있는 계정으로 로그인한 후 「사업관리」 → 「승인 대기」 메뉴로 이동합니다.

🖼
[화면 캡처 : 승인 대기 목록 화면]
승인 대기 사업 리스트

승인 처리 절차

  1. 「승인 대기」 목록에서 대상 사업 선택
  2. 사업 정보 검토 (첨부파일 포함)
  3. 「승인」 또는 「반려」 클릭
  4. 반려 시 반려 사유를 필수로 입력한 후 「확인」
  5. 승인 완료 시 담당자에게 알림 메일 자동 발송
⚠️ 주의사항
승인 후에는 되돌릴 수 없습니다. 검토가 불충분한 경우 「반려」 처리 후 담당자에게 재제출을 요청하세요.
🖼
[화면 캡처 : 승인 완료 화면]
승인 완료 후 상태 변경 확인
