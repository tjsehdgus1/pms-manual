선정결과 처리 관리자

메뉴 경로: 관리자 > 평가관리 > 평가결과

1. 평가결과 공고 목록 진입

왼쪽 메뉴 「평가관리」 → 「평가결과」 클릭 → 해당 공고 행 클릭

2. 프로그램 / 과제 목록 확인

메뉴 경로: 평가결과 > 프로그램/과제 목록

프로그램별 평가 대상 과제 목록 및 결과 현황을 확인합니다. 검토할 과제 행을 클릭합니다.

3. 평가결과 상세 확인

메뉴 경로: 평가결과 > 평가결과 상세

과제별 총점·순위·선정 여부 등 상세 결과를 확인합니다.

4. 평가위원 평가표 확인 (선정평가)

메뉴 경로: 평가결과 상세 > 평가위원 평가표 (선정평가)

각 평가위원이 제출한 항목별 점수와 의견을 확인합니다.

5. 평가위원장 평가표 확인 (선정평가)

메뉴 경로: 평가결과 상세 > 평가위원장 평가표 (선정평가)

평가위원장이 제출한 종합 평가 내용을 확인합니다.

💡 안내
선정결과 확인 후 기업에게 평가결과를 공개하려면 결과 공개 처리를 진행합니다. 이후 기업은 평가결과를 조회하고 제출할 수 있습니다.

6. 보완요청 답변 확인 후 검토완료 / 반려

메뉴 경로: 평가결과 상세 > 보완요청 및 답변 확인

기업이 보완해야 할 내용이 있을 경우 관리자가 보완요청을 진행하고 기업이 답변 시 답변내용을 확인 후 「검토완료」 또는 「반려」 처리합니다.

⚠️ 주의사항
반려 시 기업에게 반려 사유가 전달됩니다. 검토완료 처리 후에는 협약정보 등록 단계로 진행합니다.
