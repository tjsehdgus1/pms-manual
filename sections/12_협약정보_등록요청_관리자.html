협약정보 등록요청 관리자

메뉴 경로: 관리자 > 과제관리 > 전자협약

1. 전자협약 공고 목록 진입

왼쪽 메뉴 「과제관리」 → 「전자협약」 클릭 → 해당 공고 행 클릭

2. 프로그램 / 과제 목록 확인

메뉴 경로: 전자협약 > 프로그램/과제 목록

협약 대상 프로그램 및 과제 목록을 확인합니다. 해당 과제 행을 클릭합니다.

3. 협약 목록 확인

메뉴 경로: 전자협약 > 협약목록

과제의 협약 진행 현황 목록을 확인합니다. 해당 협약 행을 클릭합니다.

4. 협약정보 작성 요청

메뉴 경로: 협약 상세 > 요청내용

요청 내용을 입력하고 「저장」 버튼 클릭 → 기업에게 협약정보 등록 요청 발송

💡 안내
요청 발송 후 기업 담당자가 협약정보를 등록해야 다음 단계로 진행할 수 있습니다.
