선정평가 등록 관리자

메뉴 경로: 관리자 > 과제관리 > 평가관리 > 평가등록

1. 평가등록 공고 목록 진입

왼쪽 메뉴 「과제관리」 → 「평가관리」 → 「평가등록」 클릭 → 해당 공고 행 클릭

2. 프로그램 / 평가표 관리

메뉴 경로: 평가등록 > 프로그램/평가표 관리

평가 대상 프로그램 목록 및 평가표 연결 현황을 확인합니다.

3. 평가표 수정 (선정평가)

메뉴 경로: 평가등록 > 평가표 수정 (선정평가)

평가 항목·배점을 확인하고 필요 시 수정합니다.

평가표양식 가져오기

「평가표양식 가져오기」 클릭 → 기존 양식 선택 후 적용

4. 평가분과 추가

메뉴 경로: 평가등록 > 평가분과 추가

「평가분과 추가」 클릭 → 분과명·평가일정 입력 후 저장

평가위원 추가

「평가위원 추가」 클릭 → 팝업에서 위원 검색 후 선택

💡 안내
평가위원장은 분과 내 위원 중 1명을 지정합니다.

5. 평가기업 매칭

메뉴 경로: 평가등록 > 평가기업 매칭

분과별 평가 대상 기업을 배정합니다. 「매칭」 버튼 클릭 → 기업 선택 후 저장

⚠️ 주의사항
평가기업 매칭 완료 후 평가위원에게 평가 진행 안내가 발송됩니다.
